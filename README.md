API list for module, [HERE](https://under-tree-e.github.io/ute-core.ngx)

## Getting started

Install the package:

```shell
npm i ngx-ute-core
```

## Services and modules

-   Cookies service
-   Translate service
-   Interface resize
-   Http transfer for Sequelize + Local DB
-   Platform detector
-   Date format converter
-   Object converter
-   Time zone fixer
